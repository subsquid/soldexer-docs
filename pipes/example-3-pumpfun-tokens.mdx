This pipe is tracking [pump.fun](https://pump.fun/) token launches and computes some daily stats.

The pipe is a part of the [`sqd-pipes` repo](https://github.com/subsquid-labs/sqd-pipes). Here are the source code links:
 - [main function of the ingester](https://github.com/subsquid-labs/sqd-pipes/blob/main/packages/soldexer-pipes/src/indexers/pumpfun.ts)
 - [data stream definition](https://github.com/subsquid-labs/sqd-pipes/blob/main/packages/streams/src/streams/svm/pumpfun/pump-fun-stream.ts)
 - [SQL definition of the raw data table and the stats materialized view](https://github.com/subsquid-labs/sqd-pipes/blob/main/packages/soldexer-pipes/db/sql/pumpfun.sql)

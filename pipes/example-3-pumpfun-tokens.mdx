This pipe is tracking [pump.fun](https://pump.fun) token launches and computes some daily stats.

It is a part of the [`soldexer` repo](https://github.com/subsquid-labs/soldexer). Here are the source code links:
 - [main function of the ingester](https://github.com/subsquid-labs/soldexer/blob/main/src/indexers/pumpfun.ts)
 - [data stream definition](https://github.com/subsquid-labs/soldexer/blob/main/src/streams/pumpfun/index.ts)
 - [SQL definition of the raw data table and the stats materialized view](https://github.com/subsquid-labs/soldexer/blob/main/src/db/sql/pumpfun.sql)

---
title: "Metaplex NFT metadata"
description: "Stream metadata from Metaplex mints and store it in ClickHouse"
---

This pipe grabs the on-chain NFT metadata created using [Metaplex](https://www.metaplex.com). To achieve this it decodes the `createMetadataAccount*` and `updateMetadataAccount*` instructions of the [Metaplex program](https://solana.fm/address/metaqbxxUerdq28cj1RbAWkYQm3ybzjb6a8bt518x1s/transactions?cluster=mainnet-alpha). Some daily stats are computed on top of that.

The pipe is a part of the [`soldexer` repo](https://github.com/subsquid-labs/soldexer). Here are some source code links:
 - [main function of the ingester](https://github.com/subsquid-labs/soldexer/blob/main/src/indexers/metaplex.ts)
 - [data stream definition](https://github.com/subsquid-labs/soldexer/blob/main/src/streams/metaplex/index.ts)
 - [SQL definition of the raw data table and the stats materialized view](https://github.com/subsquid-labs/soldexer/blob/main/src/db/sql/metaplex.sql)

[Instructions for running locally](https://github.com/subsquid-labs/soldexer/tree/main?tab=readme-ov-file#getting-started)
